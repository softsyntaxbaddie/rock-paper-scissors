

##Rock, Paper, Scissors Game


A simple and interactive Python game where you play Rock, Paper, Scissors against the computer.
Track your score and see if you can beat the computer.


---

Features

Play against the computer.

Real-time score tracking.

Quit anytime by typing "quit".

Beginner-friendly Python code.

Developed and tested in VS Code.



---

Installation

1. Ensure you have Python 3 installed

2. Clone this repository or download the .py file

3. Open the project in VS Code

4. Run the game inside VS Code terminal

---

How to Play

1. Type your choice: rock, paper, or scissors.


2. The computer will randomly choose its option.


3. The game announces the winner of the round.


4. Score is updated after each round.


5. Type "quit" to end the game and see the final score.
